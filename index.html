PROCHAINE SÉANCE
Activer les vraies notifications 🔔
Les notifications push qui fonctionnent même app fermée nécessitent un Service Worker — impossible depuis un fichier HTML local.

Solution en 5 min : Héberge l'app sur GitHub Pages (gratuit) :
1. Créé un compte GitHub
2. Nouveau repo → upload bodyflow.html
3. Settings → Pages → Source: main
4. URL type : tonnom.github.io/bodyflow

En attendant, le toggle active les notifications de session active (pendant que l'app est ouverte).
